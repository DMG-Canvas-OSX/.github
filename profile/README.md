## Overview

<a href="#">DMG Canvas</a> is the purpose-built tool for creating the disk image installers that Mac application developers distribute to users. When a Mac app ships as a DMG, the window that opens after mounting — with its background image, arranged icons, and Applications folder alias — is what DMG Canvas designs and builds. Professional-looking installers communicate quality and build user trust before installation even begins.

The <a href="#">visual design editor</a> displays the exact DMG window users will see after mounting. Drag the application icon and Applications alias to their positions on the canvas. Set the background image that fills the window. <a href="#">Window dimensions</a>, toolbar visibility, icon size, and text label position all configure visually with immediate preview. What you see in the editor is precisely what users see on their Macs.

<a href="#">Background design tools</a> help create professional DMG backgrounds without leaving the application. Import existing artwork or use the built-in design canvas to create backgrounds directly. Position instructional arrows and text that guide users to drag the application to Applications. <a href="#">Retina display support</a> provides 2x resolution backgrounds that look sharp on high-DPI screens without pixelation.

The <a href="#">build system</a> compresses the finished disk image with configurable compression settings, code signs the DMG for Gatekeeper compatibility, and produces the final distributable file. <a href="#">Notarization workflow</a> integrates with Apple's notarization process, producing DMGs that macOS accepts without security warnings. These final production steps are what separate a properly distributed Mac application from one that triggers security dialogs on customer Macs.

## Key Features

- <a href="#">Visual DMG editor</a> shows the exact installer window users see with real-time drag-and-drop layout control
- <a href="#">Background image tools</a> support imported artwork or built-in canvas design for professional DMG backgrounds
- <a href="#">Retina resolution support</a> delivers sharp 2x backgrounds on high-DPI displays without any pixelation
- <a href="#">Window configuration</a> controls dimensions toolbar visibility icon size and label position visually
- <a href="#">Code signing integration</a> produces Gatekeeper-compatible disk images without triggering security warnings
- <a href="#">Notarization workflow</a> connects to Apple notarization so finished DMGs pass macOS security checks
- <a href="#">Template system</a> saves complete DMG configurations for fast rebuilds across application versions
- <a href="#">Command-line build support</a> integrates DMG creation into Xcode scripts and automated CI pipelines
- <a href="#">License agreement display</a> presents EULA text requiring acceptance before the DMG contents are accessible
